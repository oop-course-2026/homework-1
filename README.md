# homework-1

The program must be documented according to the following guidelines:

1. At the beginning of each class, include:
   - Your full name
   - Your ID number
   - A short comment explaining the purpose of the class.

2. Before each method, write a short comment explaining:
   - What the method does
   - Which parameters it receives
   - What it returns.

3. If the meaning of variables is not clear from their name, explain them using comments.

4. Add comments to explain code sections that are not trivial.

5. Do not write unnecessary comments that simply repeat what the code already clearly shows.

6. Use clear and meaningful names for variables and methods.
